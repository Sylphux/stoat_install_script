# Introduction

This is a simple shell install script for Stoat, an open source alternative for discord.

At the moment there is no support for packet managers so this script will install stoat from the release on github.

# How to use ?

## Prerequisites

Check the latest release version, update it at the top of the script before launching it.

Be sure to have installed all the dependencies and chromium beforehand.

## How to start the script


```
chmod +x stoat_install.sh
./stoat_install.sh
```
